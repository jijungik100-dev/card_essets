## 🔗 추가 카드 시너지 연동

### 💀 표식 덱 강화
기존 표식 카드(S04, M07, M10, A11, A14)에 **3장 추가**:
- **S17 벨벳**: 2명 동시 표식 + 드로우 → 표식 시딩 효율 최고
- **A27 로제**: 표식 적 2명 동시 4 데미지 → 다중 표식 활용
- **M27 아르테미아**: 위치 무관 7 데미지 + 표식 or 빙결 → 표식 체인

**강화 콤보**: S17(2명 표식) → A27(표식 적 2명에 각 4 데미지) → M27(살아남은 적에 표식+빙결)

### 🔥 화상 덱 강화
- **M28 스칼렛**: 화상 2중첩 부여 + 마법 연계 보너스 → M19 살바토레 폭발과 궁합 최고

**강화 콤보**: M03(화상 시딩) → M28(화상 2중첩 + 연계 +4) → M19(화상 폭발 ×3)

### 😱 공포 + 함정 덱 강화
- **M26 자하라**: 전체 속박 + 공포 적 추가 데미지 → 공포 덱의 중간 피니셔
- **U16 세이렌**: 적을 앞으로 끌어당김 → 함정 즉시 발동 (기존 넉백과 **반대 방향** 유도)

**강화 콤보**: T01(뒤쪽 함정) → M13(전체 공포→후퇴→함정) → U16(다시 앞으로→앞쪽 함정) → M26(속박+공포 추가 데미지)

### 🔄 핸드 어드밴티지 강화
- **U17 포춘**: 3장 드로우 후 1장 무료 사용 → 상황 대응력 극대화
- **S17 벨벳**: 드로우 + 코스트 할인 → 기존 S08 펜리스, S11 크로노와 시너지
